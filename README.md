# HIT-MHAD
## Introduction
HIT-MHAD数据集是一个包含采集自kinect azure相机的数据和拉伸传感器的数据的多模态人体活动数据集，包含来自10位实验员的24种动作。该数据集提供时间同步的深度视频数据、红外视频数据、骨骼坐标时序数据和拉伸传感器时序数据。
